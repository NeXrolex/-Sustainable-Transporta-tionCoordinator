## Project Overview
This workshop presents a systems analysis of mobility conditions in engineering faculty locations, with emphasis on parking availability, transportation habits, and shared mobility practices.

## Objective
The objective of this work is to identify system elements, mobility problems, and improvement opportunities for a future sustainable transportation coordination platform.

## Scope
The analysis focuses on Calle 40, Calle 34, and the ECCI location.

## Data Collection Methods
- Direct observation
- Physical measurements
- Structured surveys
- Process documentation

## Repository Structure
- `Workshop No.1.pdf`: final report
- `Data/`: survey files and collected data
- `Observation_Images/`: field observation photographs
- `README.md`: repository description

## Key Findings
- Parking capacity is limited in the analyzed locations.
- Calle 40 shows congestion and overflow effects during peak demand.
- Access control depends on manual security decisions.
- Informal shared mobility practices already exist among users.
